Top Albums List

API para consumo em aplicações Frontend e/ou Mobile

Modo de uso:

Método URL Descrição
GET https://bmsrangel-albums.herokuapp.com/api/album Retorna um objeto JSON com os álbuns cadastrados no banco de dados
POST https://bmsrangel-albums.herokuapp.com/api/album Adiciona um álbum ao banco de dados. Necessário enviar um objeto JSON com os atributos title, artist, url, image e thumbnail.
O atributo url deve conter o link HTTP de compra do álbum no site https://www.amazon.com/;
O atributo image deve conter o link HTTP da imagem do álbum no site https://www.amazon.com/;
O atributo thumbnail deve conter o link HTTP da imagem de perfil do artista.
